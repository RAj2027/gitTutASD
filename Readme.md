## Prerequisites

- [Python 3.x](https://www.python.org/downloads/)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/gitTutASD.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gitTutASD
   ```

## Usage

Run the main application using Python:

```bash
python app.py
```
